## 🎯 Project Overview

A modern survey form collecting user feedback on a learning platform. Features futuristic glassmorphism design with gradient backgrounds, smooth animations, and real form submission to email via FormSubmit.co.

---

## 🚀 Live Features

### Form Elements
- **Text Input:** Name field with placeholder
- **Email Input:** Email validation with required attribute
- **Number Input:** Age field (optional, min 13, max 120)
- **Dropdown:** Current role selection (Student, Learner, Job, etc.)
- **Radio Buttons:** Recommendation likelihood (Definitely, Maybe, Not sure)
- **Dropdown:** Favorite feature selection
- **Checkboxes:** Multiple improvement suggestions
- **Textarea:** Comments section with placeholder

### Form Submission
- **Action:** FormSubmit.co integration
- **Method:** POST request
- **Hidden Fields:** _captcha, _subject, _next
- **Destination:** Form data sent to email inbox
- **Confirmation:** Thank-you page after submission

---

## 🎨 Design Features

### Visual Style
- **Theme:** Futuristic tech with blue/purple gradients
- **Background:** Dark base with radial gradient overlays
- **Form Container:** Glassmorphism with backdrop blur
- **Typography:** Orbitron (headings), Rajdhani (body)

### Validation States
- **Invalid Input:** Red border on invalid unfocused fields
- **Valid Input:** Green border when valid data entered
- **Placeholder Styling:** Muted color with opacity
- **Autofill Override:** Custom styling for browser autofill

---

## 📱 Responsive Design

### Desktop
- Centered form (60vw, max 500px)
- Full-width inputs with proper spacing
- Hover effects on all interactive elements
